## Usage

Add the TOC component to any MDX page:

```mdx
## On This Page

<div class="toc-container">
  <div class="toc-header">On this page</div>
  <nav class="toc-nav">
    <a href="#overview" class="toc-item">Overview</a>
    <a href="#installation" class="toc-item">Installation</a>
  </nav>
</div>
```
