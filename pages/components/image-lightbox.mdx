# Image Lightbox

Click to expand images in a full-screen overlay.

## Example

<div class="lightbox-container">
  <div class="lightbox-preview" onclick="this.nextElementSibling.style.display='flex'">
    <div class="lightbox-placeholder">Dashboard Preview (Click to expand)</div>
  </div>
  <div class="lightbox-overlay" onclick="this.style.display='none'">
    <div class="lightbox-content">
      <div class="lightbox-image">Dashboard Preview - Full Size</div>
      <button class="lightbox-close">×</button>
      <button class="lightbox-nav lightbox-nav--prev">‹</button>
      <button class="lightbox-nav lightbox-nav--next">›</button>
    </div>
  </div>
</div>

## Usage

```html
<div class="lightbox-container">
  <div class="lightbox-preview" onclick="openLightbox(this)">
    <img src="thumbnail.jpg" alt="Description" />
  </div>
  <div class="lightbox-overlay" onclick="closeLightbox(this)">
    <div class="lightbox-content">
      <img src="full-size.jpg" alt="Description" />
      <button class="lightbox-close" onclick="closeLightbox(this)">×</button>
    </div>
  </div>
</div>
```
