# Callout Component

Styled callout boxes for important information.

## Callout Types

<div class="callout callout--info">
  <div class="callout-header">
    <span class="callout-icon">ℹ️</span>
    <strong>Info</strong>
  </div>
  <div class="callout-body">
    This is an informational message for the user.
  </div>
</div>

<div class="callout callout--tip">
  <div class="callout-header">
    <span class="callout-icon">💡</span>
    <strong>Tip</strong>
  </div>
  <div class="callout-body">
    Here's a helpful suggestion to improve your implementation.
  </div>
</div>

<div class="callout callout--warning">
  <div class="callout-header">
    <span class="callout-icon">⚠️</span>
    <strong>Warning</strong>
  </div>
  <div class="callout-body">
    Be careful with this step - it may have unintended consequences.
  </div>
</div>

<div class="callout callout--caution">
  <div class="callout-header">
    <span class="callout-icon">🔔</span>
    <strong>Caution</strong>
  </div>
  <div class="callout-body">
    This action cannot be easily undone.
  </div>
</div>

<div class="callout callout--danger">
  <div class="callout-header">
    <span class="callout-icon">🚨</span>
    <strong>Danger</strong>
  </div>
  <div class="callout-body">
    This action may cause data loss or security issues.
  </div>
</div>

## With Code

<div class="callout callout--info">
  <div class="callout-header">
    <span class="callout-icon">ℹ️</span>
    <strong>Note</strong>
  </div>
  <div class="callout-body">
    <p>You can verify your API key is working:</p>
    <pre><code>curl -H "Authorization: Bearer $ZU_API_KEY" https://api.zu.com/v1/ping</code></pre>
  </div>
</div>

## With List

<div class="callout callout--tip">
  <div class="callout-header">
    <span class="callout-icon">💡</span>

  </div>
