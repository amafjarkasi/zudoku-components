# API Parameter Table

Reference tables for API endpoint parameters and schemas.

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string | Yes | Unique identifier of the product |
| `order_id` | string | Yes | Unique identifier of the order |
| `customer_id` | string | No | Filter by customer ID |

## Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `limit` | integer | No | `20` | Number of items to return (1-100) |
| `offset` | integer | No | `0` | Pagination offset |
| `sort` | string | No | `created_at` | Sort field |
| `order` | string | No | `desc` | Sort order (`asc` or `desc`) |
| `status` | string | No | — | Filter by status |
| `created_after` | ISO 8601 | No | — | Filter by creation date |
