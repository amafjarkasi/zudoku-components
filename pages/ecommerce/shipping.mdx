## Rate Calculation

### Real-time Shipping Rates

```typescript
const rates = await client.shipping.get_rates({
  from: {
    postal_code: "97201",
    country: "US"
  },
  to: {
    postal_code: "10001",
    country: "US"
  },
  packages: [
    {
      weight: 2.5,
      dimensions: { length: 12, width: 8, height: 6 },
      package_type: "box"
    }
  ]
});
```

## Label Generation

### Printing Shipping Labels

```python
label = client.shipping.create_label(
    order_id="ord_xyz789",
    carrier="fedex",
    service="ground",
    label_format="pdf"
)
```

<Callout type="info">
  **Label Caching:** Generated labels are cached for 30 days. Reprint by referencing the `label_id` without regenerating.
</Callout>

## International Shipping

### Customs Documentation

For international orders, we automatically generate:

- Commercial Invoice (CI)
- Packing List
- Certificate of Origin (when required)
- Customs Declaration forms

```typescript
const shipment = client.shipping.create_international(
    order_id="ord_intl_456",
    customs_info={
        "duties": "delivered_duty_paid",
        "eori_number": "GB123456789",
        "export_reason": "sale"
    }
);
```

### Restricted Items

We automatically block shipment of items classified as:

- Hazardous materials (HAZMAT)
- Controlled substances
- Oversized packages exceeding carrier limits
- Items prohibited in destination country

<Callout type="info">
  **Country Restrictions:** Each destination country has specific import restrictions. Check the [Country Guide](/reference/countries) before expanding to new markets.
</Callout>

## Tracking

### Real-time Tracking Updates

```json
POST /v1/webhooks/shipping_events
{
  "event": "tracking.update",
  "tracking_number": "794644790132",
  "carrier": "fedex",
  "status": "in_transit",
  "location": "Memphis, TN",
  "timestamp": "2026-05-10T14:30:00Z",
  "estimated_delivery": "2026-05-12"
}
```

### Tracking Events Table

| Event | Description |
|---|---|
| `label_created` | Shipping label generated |
| `picked_up` | Package collected by carrier |
| `in_transit` | En route between facilities |
| `out_for_delivery` | With delivery driver |
| `delivered` | Confirmed delivery |
| `exception` | Delivery issue (weather, address) |
| `returned` | Package returning to sender |

## Returns & Reverse Logistics

### Creating Returns

```python
return_request = client.returns.create({
    "order_id": "ord_xyz789",
    "items": [
        {"variant_id": "var_sneaker_black_10", "quantity": 1}
    ],
    "reason": "wrong_size",
    "requested_action": "exchange"
})
```

## Shipping API Reference

### Endpoints

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/v1/shipping/rates` | Calculate shipping rates |
| `POST` | `/v1/shipping/labels` | Create shipping label |
| `GET` | `/v1/shipping/labels/{id}` | Get label by ID |
| `GET` | `/v1/shipping/tracking/{tracking}` | Track shipment |
| `POST` | `/v1/shipping/returns` | Create return shipment |
